# Welcome to React Cashmere

<br />
React Cashmere is a library of common components that will help aid in the development
of HealthCatalyst React based applications. This libraries core components are based
on [Material UI v5.8.6](https://mui.com/material-ui/getting-started/overview/) and
then styled using [Fiber.Cashmere](https://cashmere.healthcatalyst.net/home) mixins.

<br />
Browse example stories now by navigating to them in the left sidebar.
<br />
<span className="tip">Tip</span>
View a stories code in the `docs` tab to learn how they work.
